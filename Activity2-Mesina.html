These are the steps to Push your github respository

First lets discuss what is git?
Git is a free and open source distributed control system designed to handle
everything from small to very large projects with speed and effciency.

Here are the steps to push your Git repository


    
        1. Open Git Bash.
        
        2. Navigate to the root directory of your project.

        3. Create a folder specifically for your project
         #mkdir [webDev]
         
            Output:
            $ mkdir webDev
            $ ls-ld webDev

        4. git add README.md



        5. $ git status 
        git status command displays the state of the working directory and the staging area.
       



        5. $ git commit -m "first commit 
        For projects that already have some code files written and ready to go, the initial commit usually includes all of those code files in one big chunk.



        6. $ git branch -M main 
        A branch in Git is simply a lightweight movable pointer to one of these commits. 


        
        7. $ git remote add origin https://github.com/Poseidon0404/poseidon
         git remote add command will create a new connection record to a remote repository


       
        8. $ git push -u origin main   
        push origin is a git command that pushes a local branch(es) to a remote repository (origin).
